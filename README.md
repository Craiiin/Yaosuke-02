# 3D 三角形旋转变换可视化（Taichi 实现）
基于 Taichi 框架实现的 3D 三角形旋转变换可视化程序，核心通过 MVP（Model-View-Projection）矩阵完成 3D 顶点到 2D 屏幕坐标的变换，呈现绕 Z 轴旋转的透视效果。

## 项目介绍
本项目是计算机图形学入门经典案例，通过手动实现 MVP 矩阵变换、透视除法、视口变换等核心步骤，将 3D 空间中的三角形顶点投影到 2D 窗口，并支持按键控制旋转，直观展示 3D→2D 投影的数学原理。

### 核心功能
- 模型变换：三角形绕 Z 轴旋转（可通过 A/D 键控制旋转方向）
- 视图变换：模拟相机视角（相机固定在 Z 轴正方向）
- 透视投影：实现「近大远小」的 3D 视觉效果
- 实时渲染：通过 Taichi GUI 动态展示旋转过程

## 代码运行
1. 将代码保存为 `3d_rotation_visualization.py`
2. 终端执行：
   ```bash
   python 3d_rotation_visualization.py
   ```
3. 交互操作：
   - 按 `A` 键：三角形绕 Z 轴**顺时针**旋转
   - 按 `D` 键：三角形绕 Z 轴**逆时针**旋转
   - 按 `ESC` 键：关闭窗口，退出程序

## 核心原理
### 整体流程
```
3D 顶点（三角形）→ 模型变换（旋转）→ 视图变换（相机视角）→ 投影变换（透视）→ 透视除法（NDC）→ 视口变换（2D 屏幕）→ 绘制渲染
```

### 关键模块说明
| 函数/模块                | 功能说明                                                                 |
|--------------------------|--------------------------------------------------------------------------|
| `get_model_matrix`       | 生成绕 Z 轴旋转的模型变换矩阵，实现三角形旋转                             |
| `get_view_matrix`        | 生成视图变换矩阵，将相机平移至坐标原点（等效于场景向相机反方向平移）       |
| `get_projection_matrix`  | 生成透视投影矩阵，先将透视平截头体挤压为长方体，再缩放到 [-1,1] 标准空间   |
| `compute_transform`      | 并行计算顶点变换：组合 MVP 矩阵 → 齐次坐标变换 → 透视除法 → 视口变换       |
| `main`                   | 初始化顶点、创建 GUI 窗口、处理交互事件、调用变换函数并绘制三角形         |

### 核心数学知识点
1. **MVP 矩阵**：3D 图形学的核心，按「投影 → 视图 → 模型」顺序组合（矩阵右乘原则）
2. **齐次坐标**：将 3D 顶点扩展为 4 维向量，统一表示旋转、平移、投影变换
3. **透视除法**：将裁剪空间坐标除以齐次项，得到 [-1,1] 范围的 NDC（标准设备坐标）
4. **视口变换**：将 NDC 坐标映射到 GUI 窗口的 [0,1] 坐标空间，实现 2D 绘制

### 核心代码片段（MVP 矩阵组合）
```python
# 组合 MVP 矩阵（右乘原则：投影 → 视图 → 模型）
model = get_model_matrix(angle)
view = get_view_matrix(eye_pos)
proj = get_projection_matrix(45.0, 1.0, 0.1, 50.0)
mvp = proj @ view @ model
```

1. 增加绕 X/Y 轴的旋转控制（修改模型变换矩阵）

## 注意事项
1. 程序默认使用 CPU 后端，如需使用 GPU 加速，可将 `ti.init(arch=ti.cpu)` 改为 `ti.init(arch=ti.gpu)`（需显卡支持）
2. 确保 Taichi 版本与 Python 版本兼容，建议使用最新稳定版
3. 旋转角度每次调整 10°，可修改代码中 `angle += 10.0` 调整旋转步长
